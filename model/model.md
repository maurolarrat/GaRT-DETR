
# Model Availability

The pretrained **GaRT-DETR** model can be downloaded through the following Google Drive link:

https://drive.google.com/file/d/1NmTq7JAXJBE_IXoXXuT0ZuWGaj2RQGbU/view?usp=sharing

The provided checkpoint contains the trained model weights required for inference and experimental reproduction.

---

If you have two videos (RGB and IR) synchronized with a drone, in the style of the Anti-UAV300, use this script to generate the detection boxes in both.

https://drive.google.com/file/d/1VFPVOA_eQHwnpJsMioErRIMbwIZYiba-/view?usp=sharing

---

If you have IR ONLY, synchronized with a drone, in the style of the Anti-UAV410 or Anti-UAV600, use this script to generate the detection boxes in both IR and PSEUDO-RGB, or only IR in both .
